# RKShoppingHub

RKShoppingHub is a high-performance e-commerce platform built to demonstrate advanced React state management and responsive UI design. This React-based application features a custom local product database, independent authentication, and a fully functional shopping cart.

✨ Key Features

Dynamic Product Catalog: Browsing 47+ unique items powered by a custom local database.

Smart Shopping Cart: Seamlessly add, remove, and update quantities with real-time price calculations.

Modern UI/UX: Fully responsive design optimized for mobile, tablet, and desktop views.

Global State Management: Uses React Context API to track cart data across the entire app.
